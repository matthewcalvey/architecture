AI Render Services

  
Available Rendering APIs

  
These AI services can transform the 3D massing model into photorealistic architectural renders. Services marked structure-preserving use the canvas screenshot as a structural guide, keeping the building geometry while adding materials and context.


  


    

      
Stability AI · Recommended

      
Structure-preserving. The control/structure endpoint sends your canvas screenshot as a structural guide. The AI preserves the building massing while adding photorealistic materials. control_strength parameter (0.01–1.0) controls geometry fidelity.

      
$0.03–$0.08/image · API key from platform.stability.ai

    


    

      
Replicate FLUX

      
Depth-aware. The flux-depth-pro model accepts a depth map from the THREE.js scene, preserving exact 3D geometry. Also supports flux-canny-pro for edge/wireframe input.

      
~$0.05/image · API key from replicate.com

    


    

      
fal.ai FLUX

      
ControlNet + LoRA. FLUX with multiple ControlNet modes (depth, canny, structure) in a single unified endpoint. Good JavaScript SDK.

      
~$0.03–$0.075/image · API key from fal.ai

    


    

      
mnml.ai

      
Architecture-specific. Purpose-built for architectural rendering with /sketch-to-img and /exterior endpoints. Trained on architectural imagery. 30% student discount.

      
~$0.10–$0.29/image · API at mnmlai.dev

    


    

      
Pollinations · Free

      
Text-to-image only. Free, no API key needed. Uses FLUX/Turbo models. Does not use the 3D massing as input — generates from text prompt only. Good for quick concepts.

      
Free · No key · pollinations.ai

    


    

      
OpenAI GPT-Image

      
Image editing. Send screenshot + prompt for general-purpose image editing. Less structural control than ControlNet-based options, but understands architecture well.

      
$0.02–$0.19/image · platform.openai.com

    


  


  

    
Integration Note

    
Paid API services require a server-side proxy to keep API keys secret (e.g. a Cloudflare Worker or Vercel Edge Function, free tier). Select your service above and paste your API key to use structure-preserving renders directly from this page. Pollinations works without any setup.
