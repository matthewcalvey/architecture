01 — Notes

Import Guidance

Rhino: Import DXF; set model units to meters. Geometry comes in at true scale in UTM.

Revit: Import CAD (DXF) using meters. If you need Shared Coordinates, set project location based on EPSG shown in the UI (sidecar PRJ in the ZIP).
